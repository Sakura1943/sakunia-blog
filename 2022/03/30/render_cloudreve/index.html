使用Render部署Serverless Cloudreve

准备工作
github 账号一个
render 账号 注册

开始

1. 注册数据库

我这里使用sqlpub注册

打开 链接
按照要求注册完数据库，然后将信息保存起来

2. 打开项目部署

登录render账号: https://dashboard.render.com/

打开Github项目链接: 打开链接

点击部署按钮:

填写数据库环境变量

点击apply完成部署

3. 查看初始化密码

点击dashbord跳转到管理面板
然后打开项目名

等待初始化完毕得到账号密码

4. 域名代理

打开你的ns厂商域名解析列表， 如我使用的 CloudFlare
用CNAME解析一个你想要的域名

CNAME的值是render提供域名， 如图
记得， 如果是cloudlflare, 请不要打开代理功能

点击添加域名， 将域名填入后应用， 然后等待状态为绿色即可

5. 登陆并修改密码

使用你的域名进行访问


邮箱和密码为上面生成的初始化邮箱和密码

打开设置

点击修改密码


输入初始化的密码和新密码

结束

关于 Cloudreve如何绑定存储介质 与 更多玩法
详情请看Cloudreve官方文档
地址: https://docs.cloudreve.org
